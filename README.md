# Wells Fargo Software Engineering Virtual Experience

This repository contains my solutions for the Wells Fargo Software Engineering Virtual Experience Program on Forage.

## Tasks Completed
- Designed an Entity Relationship Diagram (ERD) for an Investment Management System.
- Implemented the data model using Java Persistence API (JPA).
- Applied entity relationships, primary keys, foreign keys, and object-relational mapping concepts.

## Technologies Used
- Java
- Spring Boot
- JPA/Hibernate
- MySQL (Relational Database)
